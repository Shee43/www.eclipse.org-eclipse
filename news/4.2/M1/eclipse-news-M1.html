Eclipse Juno M1 New and Noteworthy

Here are the noteworthy things available in milestone build M1 (August 5, 2011) which is now available for download.

Equinox and Platform
Git migration During M1 the Equinox and Platform teams were busy with the Git migration. So far the following got migrated:
  • Equinox
  • SWT
  • Platform Runtime
  • Platform UI
JDT
Java 7 support Promptly after Java 7 has been officially released, Eclipse also fully supports it. See What's new for Java 7 for the new Java 7 related features offered by JDT.
PDE
Plug-in vendor history When creating a new plug-in, fragment or feature the Vendor information used in the creation wizard is retained and can be reused by selecting the drop-down in subsequent uses of the wizard.
