Welcome to the Eclipse SDK project!

The Eclipse SDK project is part of Photon, the Eclipse Foundation's thirteenth yearly Simultaneous Release, available June 27, 2018. The Eclipse SDK and related resources can be downloaded from the Eclipse Project downloads page.

Here are some of the more noteworthy items available in the eighteenth annual release of the Eclipse SDK project.

  • New features in the Platform
  • New features for Java developers
  • New APIs in the Platform and Equinox
  • New features for plug-in developers
