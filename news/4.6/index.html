Eclipse Neon - New and Noteworthy

Neon.
From Greek νέον: new, recent

Welcome to Neon, the Eclipse Foundation's eleventh yearly Simultaneous Release, available June 22, 2016. It includes content from eighty-five Eclipse projects.

The Eclipse platform and related resources can be downloaded from the Eclipse Project downloads page.

Here are some of the more noteworthy things available in the sixteenth annual release of the Eclipse project SDK.

  • New features in the Platform and Equinox
  • New features for Java developers
  • New APIs in the Platform and Equinox
  • New features for plug-in developers

And don't miss the news for the Neon.1 and Neon.2 update releases!
Neon.3 was a bug-fix release without notable new features.
