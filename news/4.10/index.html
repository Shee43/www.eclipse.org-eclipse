Eclipse 4.10 - New and Noteworthy

Welcome to the Eclipse SDK project!

The Eclipse SDK project is part of the Eclipse 2018-12 simultaneous release, available December 19, 2018. The Eclipse SDK and related resources can be downloaded from the Eclipse project 4.10 downloads page. The Eclipse installer and other packages can be downloaded from the Eclipse Installer page.

Here are some of the more noteworthy items available in this release.

  • New features in the Platform and Equinox
  • New features for Java developers
  • New APIs in the Platform and Equinox
  • New features for plug-in developers
